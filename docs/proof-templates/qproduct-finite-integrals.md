# QProduct Finite Integrals

Use q-product finite integral certificates when the statement is an exact finite
product-integral identity.

This is the finite layer:

```text
finite q-product
+ exact rational moment/integral computation
= product-integral theorem
```

Prime-limit certificates add monotone/sandwich and tail arguments on top of the
finite layer; see [Domain Libraries → QProduct Prime Limits](../domains/qproduct-prime-limits.md).

Detailed API reference: [QProduct Certificates](../certificates/qproduct.md).
