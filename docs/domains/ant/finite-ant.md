# Finite ANT Certificates

Finite ANT certificates cover exact finite arithmetic transforms:

- step sums;
- Abel summation;
- Euler and log products;
- Dirichlet truncations;
- Mertens-style finite sums;
- prime-power extensionality wrappers.

These are domain-specific packages built on the checker-to-theorem pattern.
